
###### 如何提问和回答问题

```
提问：What + do + you + verb?
 
表达你喜欢、喜爱、厌恶： I + verb + -ing。
 
Example Question: What do you like? Example Answer: I like shopping.
示例问题： 你喜欢什么？ 示例回答：我喜欢购物。
```
